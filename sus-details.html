Suspect User Simulator is a real-time simulation inspired in the game Among Us, where the workers and impostors are independent AI agents using Finite State Machines, Behaviour Trees and Utility Systems.

My labour in this project was among various implementation issues such as designing, coding and setting up the Unity project for the simulation to be fully working and customizable.
